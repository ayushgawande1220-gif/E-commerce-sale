# E-Commerce Sales Analysis (Excel Project)

# Project Overview
This project presents a complete **E-Commerce Sales Analysis** using Microsoft Excel.  
It includes **data cleaning, analysis, and interactive dashboard creation** to generate meaningful business insights.

# Objective
The goal of this project is to:
- Analyze sales and profit performance  
- Identify top-performing products and categories  
- Understand the impact of discounts on profit  
- Build an interactive dashboard for decision-making  

# Dataset Details
The dataset contains:
- Order ID  
- Order Date  
- Customer Name  
- Product Name  
- Category  
- Sales  
- Quantity  
- Discount  
- Profit

# Key Insights
- High discounts reduce profit margins  
- Few products generate most revenue (Pareto principle)  
- Some categories have high sales but low profit  
- Sales show variation across time and regions
